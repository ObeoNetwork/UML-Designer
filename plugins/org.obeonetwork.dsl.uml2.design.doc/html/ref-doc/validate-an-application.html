Validate an application Reference Documentation

  1. Home
  2. Validate an application

UML Designer provides a mean to evaluate and ensure the well-formedness of the UML model. On any diagram a tool is available to launch diagram validation.

The validation errors and warnings are listed in the problem view and decorators are positioned on the invalid elements in the diagrams.
