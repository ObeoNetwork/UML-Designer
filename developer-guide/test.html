Tests developer guide

Unit tests

The unit and plugin tests are defined in the org.obeonetwork.dsl.uml2.design.tests project.
These tests are run automatically by the build.

UI tests

The UI tests are defined thanks to RCPTT.
The tests are available on another github repository : https://github.com/ObeoNetwork/UML-Designer-UI-Tests

The UI tests are built thanks to a specific UI Tests build.
